Пример: Измерение температуры с Arduino

Код программы:

Данный пример демонстрирует использование датчика температуры TMP36 для измерения температуры и вывода ее значения через последовательный порт. Код программы инициализирует аналоговый пин A0 в качестве входа для считывания значения напряжения с датчика. Затем происходит преобразование напряжения в температуру в градусах Цельсия, и результат выводится через последовательный порт с помощью функции Serial.print().

Схема подключения:

В данном случае, датчик температуры TMP36 подключается к аналоговому пину A0 на Arduino. Контакт VCC датчика подключается к 5V пину Arduino для обеспечения питания, контакт GND подключается к GND пину Arduino, а контакт OUT подключается к аналоговому пину A0. После подключения датчика Arduino будет считывать напряжение с датчика, которое затем будет преобразовано в температуру.
