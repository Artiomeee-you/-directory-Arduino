Пример 2: Управление яркостью светодиода с помощью потенциометра

Код программы:

В данном примере используется потенциометр (резистор с переменным сопротивлением) для управления яркостью светодиода. При вращении потенциометра меняется его сопротивление, и это значение считывается в аналоговом входном пине Arduino с помощью функции analogRead(). Затем считанное значение преобразуется в диапазон яркости светодиода (от 0 до 255) с помощью функции map(), и устанавливается с помощью функции analogWrite() на выходном пине, к которому подключен светодиод.

Схема подключения:

В данной схеме потенциометр подключается к одному из аналоговых входных пинов (например, A0) Arduino. Один конец потенциометра подключается к напряжению питания платы Arduino (5V), другой конец - к земле (GND), а серединка - к выбранному аналоговому входному пину. Светодиод, который управляется яркостью, подключается к выбранному выходному пину, к которому также подключается резистор с ограничивающим сопротивлением для контроля тока.
